APPLICATION SCREENSHOTS
======================

NOTE: .txt files cannot display images. See SCREENSHOTS.md for visual documentation.

1. REGISTRATION PAGE
   - Location: /register or click "Don't have an account? Register here"
   - Image: screenshots/register-page.png
   - Features: Email input, password input with toggle visibility, create account button
   - Description: Clean registration form with validation

2. LOGIN PAGE
   - Location: /login
   - Image: screenshots/login-page.png
   - Features: Email/password login, link to register page
   - Description: User authentication interface

3. DASHBOARD PAGE
   - Location: /dashboard (protected route)
   - Image: screenshots/dashboard.png
   - Features: Factory metrics, worker table, workstation table, seed data button
   - Description: Main analytics dashboard after login

HOW TO ADD SCREENSHOTS:
-----------------------
1. Create a 'screenshots' folder in the project root
2. Save your screenshots with descriptive names (e.g., register-page.png, login-page.png, dashboard.png)
3. Reference them in SCREENSHOTS.md using markdown image syntax: ![Description](screenshots/image-name.png)
